  • Redis 常见误解
    • Redis 占用的内存比关系式数据库要多
    • Redis 和 Memcached 的功能一样
    • Redis 会丢失数据，所以它只能作为缓存使用，而不能作为数据库
    • Redis 不能执行复杂的操作，比如反向搜索
    • Redis 不能保存和处理结构化数据，比如 JSON
    • Redis 是一个单机数据库
